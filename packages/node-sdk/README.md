# @byfriends/sdk

The TypeScript SDK for BYF.

Part of the [BYF](https://github.com/ByronFinn/byf) monorepo.

**Runtime:** Bun-only (import and run under Bun). Node.js is not a supported interpreter for this library. See [ADR 0028](../../docs/adr/0028-full-bun-toolchain.md). End users of the CLI should install `@byfriends/cli` (native binary), not this package, unless embedding the SDK.

See the main repository for documentation, issues, and contribution guidelines.
