# @byfriends/cli-darwin-arm64

Platform binary package for [`@byfriends/cli`](https://www.npmjs.com/package/@byfriends/cli) on **darwin-arm64**.

Do not install this package directly. Install the main package:

```bash
npm install -g @byfriends/cli
```

The binary is produced by the same `bun build --compile` pipeline as GitHub Release assets.
